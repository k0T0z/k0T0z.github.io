This blog post is related to my GSoC 2023 Project.

Finally, GameDev.tv's Game Jam 2023 is over and as GameDev.tv promised, any submission will get a free course from their list. I submitted 4 simple games to the competition and got 3 free courses. I picked:

  • Unreal Engine 5 C++ Developer: Learn C++ & Make Video Games.
  • Unreal C++ Multiplayer Master: Intermediate Game Development.
  • C++ Fundamentals: Game Programming For Beginners.

You might be wondering, how these courses will benefit me in GSoC? Well, Not all of them will benefit me in GSoC, but the multiplayer courses will help me to understand more about Steamworks API in action. In section 3, in the Unreal C++ Multiplayer Master: Intermediate Game Development course, I will learn how to integrate Steamworks API to create a multiplayer game. I actually didn't watch the whole section due to my finals, but I will watch it after my finals, then I will answer the question how? in a future blog post.

Although I have final exams running, I started implemented the Overlay API, check my Midterm Evaluation PR, and I actually finished some of the General API which helped me figured out that I need to test all those APIs in a game, no way I can just implement them and leave them like this because although ENIGMA is based on GameMaker, it's not GameMaker, and I need to make sure that all those APIs are working as expected. Decision is made, I will create a game to test all those APIs, and I will place it in my Steamworks extension directory /enigma-dev/ENIGMAsystem/SHELL/Universal_System/Extensions/Steamworks/.
